# 需求边界文档

## 项目定位

| 项目 | 内容 |
|------|------|
| **类型** | 独立积分兑换商城 |
| **部署模式** | 多套独立部署（每个娱乐城一套） |
| **商城名称** | 可自定义配置 |

## 技术选型

| 项目 | 内容 |
|------|------|
| **参考源码** | macrozheng/mall |
| **后端** | Spring Boot + MyBatis + MySQL + Redis + ES + RabbitMQ |
| **前台框架** | uni-app（样式参考 lhshop.online）|
| **后台框架** | Vue + Element UI |
| **部署** | Docker 容器化 |
| **环境** | 测试/正式分离，正式环境需授权更新 |

## 功能范围

### 前台功能（参考 lhshop.online）
- 首页（爆款推荐 + 分类展示）
- 商品分类浏览
- 商品列表 / 商品详情
- 购物车
- 订单流程
- 我的订单
- 会员中心
- 公告 / 新闻

### 后台功能（保留 mall 定义）
- 商品管理
- 订单管理
- 会员管理
- 促销管理
- 运营管理
- 内容管理
- 统计报表
- 财务管理
- 权限管理

### 商品类型
- 实体商品（手表、包包、电器等）
- 虚拟商品（彩金、超商卡）

## 娱乐城对接

### 货币单位
点数（从娱乐城帐号获取）

### 需要对接的 API

| 接口 | 用途 | 提供方 |
|------|------|--------|
| 用户验证 | 登入验证娱乐城帐号 | 娱乐城 |
| 点数查询 | 显示可用点数 | 娱乐城 |
| 点数扣除 | 下单扣点 | 娱乐城 |
| 彩金发放 | 虚拟商品兑换加余额 | 娱乐城 |

### 对接状态
⏳ 待对接（API 文档待提供）

## 可配置项（多套部署）

每套商城可独立配置：
- 商城名称
- 商城 Logo
- 娱乐城 API 端点
- 数据库连接
- Redis 连接
- 其他品牌相关设置
