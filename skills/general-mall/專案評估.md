# 商城系統 - 專案評估

## 專案概述

一般電商平台，基於 mall 完整功能，支援多套獨立部署。

---

## 技術架構

| 項目 | 規格 |
|------|------|
| 參考源碼 | macrozheng/mall（完整保留） |
| 後端 | Spring Boot + MyBatis + MySQL + Redis + ES + RabbitMQ |
| 前台框架 | uni-app（原版） |
| 後台框架 | Vue + Element UI |
| 部署 | Docker + AWS RDS + ALB |
| 多套部署 | ✅ 商城名稱/Logo 可配置 |

---
